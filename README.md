
## 
Esta aplicación fue realizada con:
- NextJS
- NodeJS
- ExpressJS
- GraphQL / Express GraphQL / Apollo Client
- TypeScript
- PostgresQl
- TypeOrm
### ¿Cómo se configura?

## FRONTEND  
1. Crear folder vacío
2. clonar el repositorio
4. git flow init
6. Rama develop
5. navegar hasta la carpeta frontend/pandashop
6. yarn
6. yarn dev
## BACKEND 

1. navegar hasta la carpeta app/backend
2. yarn
3. yarn dev
